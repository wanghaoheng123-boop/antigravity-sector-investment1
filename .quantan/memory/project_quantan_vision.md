QUANTAN is being developed as a comprehensive trading/investment intelligence platform covering global stocks, commodities, options, and crypto.

**Why:** The user wants a single platform that provides high-confidence entry/exit signals for both short-term trades and long-term investments, functioning like a Bloomberg terminal but accessible. Target: >80% signal accuracy across all market conditions.

**How to apply:** All feature additions should prioritize signal accuracy and backtesting rigor. New algorithms must be backtested before deployment. The 7-phase upgrade plan is documented in `AGENTS.md`. Phase 1 (testing foundation) must complete before any other phase begins.

**Current baseline:** 56.35% win rate on 56-instrument backtest (post Phase 2 signal hardening).
